End-to-End App Testing:

TEST SCENARIOS – DEMOBLAZE (PRODUCT STORE)

TS1: Sign Up:
----------
TC1 Valid Username and Valid Password → System should display "Sign up successful" alert
TC2 Valid Username and Blank Password → Alert should be displayed (no strict validation in app) [Please fill out Username and Password.]
TC3 Blank Username and Valid Password → Alert should be displayed
TC4 Blank Username and Blank Password → Alert should be displayed

TC5 Existing Username → System should display "This user already exist"
TC6 Existing Username with different password → Same alert should be displayed
TC7 New Username with any password → Sign up should be successful

TC8 Click Close button → Sign up modal should close
TC9 Click Cross (X) → Sign up modal should close

Validation:
Alert message text should match expected
Modal should open and close correctly

TS2: Log in:
----------
TC1 Valid Username and Valid Password → User should be logged in successfully
TC2 Valid Username and Invalid Password → System should display "Wrong password" alert
TC3 Invalid Username and Valid Password → System should display "User does not exist"
TC4 Invalid Username and Invalid Password → Error alert should be displayed

TC5 Blank Username and Password → Alert should be displayed (no strict validation)

TC6 Click Close button → Login modal should close
TC7 Click Cross (X) → Login modal should close

TC8 Successful login → Username should be displayed on navigation bar
TC9 Click Logout → User should be logged out and username should disappear

Validation:
Username should be visible after login
Alert messages should match expected
Session should be updated after logout

TS3: About Us:
----------
TC1 Click About Us → About Us modal should open
TC2 Video section should be visible inside modal
TC3 Click Play → Video should start (may vary based on browser behavior)

TC4 Click Close button → Modal should close
TC5 Click Cross (X) → Modal should close

Validation:
Modal visibility should be correct
Video iframe should be loaded

TS4: Contact:
---------
TC1 Click Contact → Contact modal should open

TC2 Enter Email, Name, Message and click Send → System should display "Thanks for the message" alert
TC3 Click Send without entering data → Same alert should be displayed (no validation in app)

TC4 Click Close button → Modal should close
TC5 Click Cross (X) → Modal should close

Validation:
Alert message should be displayed
Modal behavior should be correct

------------
TC1 Login with valid credentials
TC2 Click Logout
